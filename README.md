# Лабораторная №1, ПнаЯВУ, группа 334701, Вариант №18

Этот проект представляет собой RESTful API, созданное с использованием Spring Boot, которое позволяет получать информацию о фильмах по их названию, используя Open Movie Database API.

Вы можете использовать Postman для отправки запросов к API.

**Метод**: `GET`

**URL**:
http://localhost:8080/movie?title={название_фильма}

**Пример запроса в Postman**:

1. Откройте Postman.
2. Выберите метод `GET`.
3. Вставьте следующий URL в адресную строку, заменив `{название_фильма}` на название фильма, который вы хотите найти. Например: http://localhost:8080/movie?title=Interstellar
